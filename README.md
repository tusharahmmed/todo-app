
# TODO APP

A TODO APP using `React.js`


## Demo

[Live Preview](https://todoapp-tushar.netlify.app/)


## Features

- Add Task
- Delete Task
- Update Status
- Trash List
- Used Localstorage as Database


## Run Locally

Clone the project

```bash
  git clone https://github.com/tusharahmmed/todo-app.git
```

Go to the project directory

```bash
  cd todo-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
